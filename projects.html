NetTech: Fog Netting System

An Arduino-based system that automates fog exposure for “Capsicum frutescens” (Siling Labuyo). Helps increase productivity and reduce pest-related losses using sensor-triggered net movement and misting.

This project simulates a smart traffic control system using Arduino, Ultrasonic sensors, and LEDs. Designed for educational purposes to demonstrate real-time traffic logic with pedestrian control.
